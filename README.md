# To-Do App

A simple and clean To-Do List web application built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as complete
- Data saved using local storage

## Tech Used
- HTML
- CSS
- JavaScript
